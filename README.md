# [✦] Installation
  <h2> Linux </h2>  

  - <code>gitclone https://github.com/Menatic007/Menatic-Buster</code>
  - <code>cd Menatic-Buster</code>
  - <code>pip install requirements.txt</code>
  - <code>sudo chmod +x MenaticBuster</code>
  - <code>sudo ./MenaticBuster</code>
